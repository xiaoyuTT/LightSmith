# Scripts 目录

此目录包含实用脚本和手动测试工具。

## 📂 文件说明

### `test_api.py` - 手动集成测试

**用途**：手动测试 API 端点的真实 HTTP 行为

**使用方法**：
```bash
# 1. 启动服务（终端1）
uvicorn app.main:app --reload --port 8000

# 2. 运行测试脚本（终端2）
python scripts/test_api.py
```

**测试内容**：
- ✅ 健康检查端点
- ✅ 单个 Run 摄入
- ✅ 批量摄入（5 个 Run）
- ✅ 幂等性验证（重复提交）
- ✅ 错误 Run 处理
- ✅ 输入验证（无效 run_type）

**与单元测试的区别**：

| 特性 | 单元测试（pytest） | 手动测试（scripts） |
|------|------------------|-------------------|
| **执行方式** | `pytest tests/` | `python scripts/test_api.py` |
| **需要启动服务** | ❌ 否（使用 TestClient） | ✅ 是（真实 HTTP 请求） |
| **测试对象** | 单个模块/函数 | 完整的 HTTP 流程 |
| **速度** | 快（~1s） | 慢（需要启动服务） |
| **用途** | 自动化测试、CI/CD | 手动验证、集成测试 |

**何时使用**：
- 验证服务部署后的实际行为
- 调试 HTTP 层问题（CORS、headers 等）
- 快速验证新功能是否正常工作
- 生成真实的请求示例（用于文档）
